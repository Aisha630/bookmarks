Welcome to BookSwitch!

Thanks for installing! This extension makes accessing your bookmarks faster than ever.

How to Use

Simply use the following keyboard shortcuts to open the corresponding bookmark from your Bookmarks Bar in a new tab.:

  • Alt+1 – Opens the 1st bookmark.
  • Alt+2 – Opens the 2nd bookmark.
  • ...up to Alt+9.

Shortcut Limit

Due to a Chrome limitation, only the first 4 shortcuts are preset (Alt + 1–4). To use shortcuts 5–9, you'll need to set them manually.

Go to chrome://extensions/shortcuts and assign keys for open-bookmark-5 through open-bookmark-9.

Customize Anywhere

If you prefer different shortcuts, change them anytime via the same extensions’ shortcuts page.

Happy browsing!
